Meteorite Project

Project Summary:

For this project, we created a web page that displayed 30,000 plus datasets of specific meteorites. The dataset comes from a CSV file provided by catalog.data.gov. The file includes each meteorite's id, name, location (specifically by latitude and longitude), discovery, and time. After cleaning up the file, we then imported the data into an SQLite database file. Once we built the database, we then worked on the HTML web page and PHP code. The PHP code implemented the database and filter options for the web page; the filtering option is based on discovery, year, or weight (maximum or minimum). Lastly, we worked on making the home page and CSS stylesheets.
